ARLive Admin Setup
Connect your Supabase project. This is saved on this device only and only needs to be done once.
You need a Supabase project with:
• Storage bucket named ar-assets (Public)
• Table named ar_targets
• RLS disabled on ar_targets
Quick Setup:
1. Go to supabase.com → New project
2. Table Editor → New table: ar_targets — disable RLS
  Columns: id (int8 pk), name (text), image_url (text), video_url (text), mind_url (text), created_at (timestamptz default now())
3. Run in SQL Editor: GRANT ALL ON TABLE public.ar_targets TO anon;
  Also run: GRANT USAGE ON SCHEMA public TO anon;
4. Storage → New bucket: ar-assets → Public ON
5. Storage Policies → New policy → anon → SELECT+INSERT+UPDATE+DELETE → expression: true
6. Settings → API → copy URL + anon key
