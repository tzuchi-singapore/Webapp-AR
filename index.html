Connect your Supabase project to get started
ℹ️ Your credentials are saved locally on this device only. You need a Supabase project with a storage bucket named ar-assets (public) and a table named ar_targets.
Quick Setup (5 min):
1. Go to supabase.com → New project
2. Table editor → New table: ar_targets
  Columns: id (int8, pk), name (text), image_url (text), video_url (text), mind_url (text), created_at (timestamptz)
3. Storage → New bucket: ar-assets → set as Public
4. Storage → Policies → Add policy: Allow all operations for anon
5. Settings → API → copy URL + anon key above
